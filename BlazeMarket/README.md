# 🔥 BlazeMarket - SMM Services Bot & Web App

**BlazeMarket** - это современный SMM маркетплейс с дизайном в стиле **Lime + Graphite**, который объединяет покупателей и продавцов SMM услуг.

## ✨ Особенности

### Для покупателей:
- 📋 **У меня есть заказ** - ввод кода заказа и отправка комментариев
- 🛒 **Я хочу купить** - выбор услуг по категориям соцсетей
- 💳 Оплата через Pally.info
- 📱 Telegram Web App интерфейс

### Для продавцов:
- 💰 **Купить бота** за 1000₽
- 🏪 **Создать магазин бесплатно** с персональным зеркалом
- ⚙️ **Настроить наценку** на услуги
- 🔗 Получить ссылку на своё зеркало

### Система зеркал:
- Каждый продавец получает уникальное зеркало
- Возможность установить свою наценку
- Пример распределения: базовая цена 5₽, наценка 100% = 10₽ клиенту, из них 8₽ владельцу, 2₽ продавцу

## 🎨 Дизайн

Стиль **Lime + Graphite**:
- Основной цвет: #32CD32 (Lime)
- Вторичный цвет: #2F4F4F (Graphite)
- Акцентный цвет: #7FFF00 (Lime Light)

## 📦 Установка

```bash
cd BlazeMarket
pip install -r requirements.txt
```

## ⚙️ Настройка

1. Скопируйте `config.json.example` в `config.json`
2. Заполните необходимые параметры:

```json
{
    "bot": {
        "token": "YOUR_BOT_TOKEN",
        "admin_ids": [YOUR_TELEGRAM_ID]
    },
    "payment": {
        "pally_merchant_id": "YOUR_MERCHANT_ID",
        "pally_secret_key": "YOUR_SECRET_KEY"
    },
    "api": {
        "twiboost_api_key": "YOUR_TWIBOOST_KEY",
        "smmway_api_key": "YOUR_SMMWAY_KEY"
    }
}
```

## 🚀 Запуск

### Telegram бот:

## 📱 Telegram Web App

Для использования Web App:
1. Откройте @BotFather
2. Создайте нового бота или выберите существующего
3. Перейдите в Menu Button → Configure Menu Button
4. Укажите URL вашего Web App: `https://your-domain.com/`

## 📝 Процесс работы

### Покупатель с заказом:
1. Выбирает "У меня есть заказ"
2. Вводит код заказа
3. Отправляет комментарии (текстом или файлом, без #)
4. Заказ отправляется в работу

### Покупатель хочет купить:
1. Выбирает категорию соцсети
2. Выбирает услугу
3. Оплачивает через Pally
4. Получает услугу

### Продавец:
1. Создаёт бесплатное зеркало
2. Настраивает наценку
3. Получает ссылку на магазин
4. Привлекает клиентов
5. Получает процент с продаж
